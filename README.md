# طبيبي — Vercel General Health App

نسخة Vercel جاهزة للرفع على GitHub ثم Vercel.

## الميزات الجديدة

- تسجيل حساب كمراجع أو طبيب.
- التسجيل عبر البريد الإلكتروني أو رقم الهاتف.
- تحقق طبيب تجريبي عبر Doctor ID.
- تطبيق عام لكل المرضى، وليس مخصصًا للكلى والضغط فقط.
- مكتبة أمراض ومتلازمات واسعة: 153 مرضًا/متلازمة.
- عند اختيار المريض أمراضه داخل الملف الصحي، تظهر له في الدليل أولًا ضمن “خاص بملفك”.
- كل مرض يفتح صفحة خاصة تحتوي على:
  - ما يساعد المريض.
  - ما يجب تجنبه.
  - علامات الخطر.
  - أسئلة للطبيب.
  - أطعمة آمنة/مناسبة.
  - أطعمة سيئة أو خطرة.
  - قواعد غذائية.
  - وصفات فطور وغداء وعشاء.

## التشغيل محليًا

```bash
npm install
npm run build
```

## إعدادات Vercel

- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## ملاحظة طبية

المحتوى للتثقيف والتنظيم فقط وليس تشخيصًا أو بديلًا عن الطبيب. حالات الطوارئ يجب التعامل معها عبر الإسعاف أو أقرب مركز صحي.
